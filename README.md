# Personal Portfolio Website

A responsive personal portfolio website built as part of the **Responsive Web Design** course at **Information Technology Institute (ITI)**.

The project focuses on converting a professional **Figma design** into clean, responsive code using modern front-end tools.

---

## 🚀 Live Demo
🔗 https://hajarzain222.github.io/Personal-Portfolio-Template/

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Flexbox
- CSS Grid
- Font Awesome

---

## 🎯 Project Goal
The goal of this project was to practice building a **fully responsive website** and translating a **Figma UI design into a functional web page** while maintaining clean and structured code.

---

## ✨ Features
- Responsive layout for all screen sizes
- Modern and clean UI design
- Bootstrap grid system
- Responsive navigation bar
- Testimonials section
- Portfolio / Recent Work carousel
- Contact form section

---

## 📚 What I Learned
Through this project I practiced:

- Using **Bootstrap components and utilities**
- Converting **Figma designs into code**
- Writing **clean and maintainable HTML & CSS**
